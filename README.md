## Installation

```bash
$ yarn install
```

## Dependencies

- Consul

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
npm run start:prod
```

After run the starter app, you can visit this urls in browser:

http://localhost:3000

http://localhost:3000/github

http://localhost:3000/proxy/github
